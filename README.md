## How to Use

1. Clone the repository:
    ```bash
    git clone https://github.com/CodeNinjaSarthak/health-check.git
    ```
   
2. Navigate to the directory:
    ```bash
    cd health-check
    ```

3. Run the `health-checks.py` script:
    ```bash
    python3 health-checks.py
    ```
